Smart Attendance System

  • Tech Stack: Python, HTML, CSS, tkinter, openCV , CSV ,Numpy
  • Github URL: Project Link

Created a attendance system which takes attendance by using face recognition technique, and intergrated it with GUI (Graphical user interface) so it can be easy to use by anyone.GUI for this project is also made on python using tkinter.

Incorporated password-protected functionalities for system access and profile saving,Encrypted password storage to enhance data security.

Implemented a training module to create a face recognition model using LBPH (Local Binary Pattern Histogram) algorithm , Utilized the trained model for real-time face recognition during attendance tracking.
